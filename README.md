# CYBEROLL

The front-end repo of rollingsagas ([dev domain](https://dev.cyberoll.pages.dev/)).

- [dev](https://dev.cyberoll.pages.dev/)

1. Install dependencies: `pnpm i`
2. Create `.env` file, and input secres here.
